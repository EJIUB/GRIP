# GRIP
The GRIP model infers gene regulation by considering TF physical proximity.

# How to run
The /Example folder includes the Jupiter notebook to reproduce GRIP results on CD4, CD8 and CD14 uing PBMC 3k data. The rest of the folders are the PQN codes that are needed to run GRIP.
